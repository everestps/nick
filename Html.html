The two most important resources to build a website are html and css. Altough both of these programs help you, they do so in different ways.

Html


Html acts as the main setup for your website. it is responsible for text, images, links, and a-lot more. Html stands for Hypertext-Markup Language. Html is basically just a standard and conveniant system for tagging. A tag is a command that changes the layout of a webpage, for example if I wanted to create a line break I would type br / inside of two pound signs. Html formating is important because it is the main makeup of a webpage. Without the html, nothing would be functional.

CSS


CSS is another crucial part into making a webpage. without CSS without it, a webpage would look bare and boring, even colorless. CSS literally means cascading style sheets, in other words, CSS is what makes the webpage look good. CSS is written on a different document than html, a link to it has to be written on the html that looks like link rel="stylesheet" type="text/css" href="Html.css" written inside pound signs. Together Html and css make a pretty good loking webpage.
